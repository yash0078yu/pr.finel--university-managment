# University Course Management System

# Project Overview
This project is a complete SQL-based University Course Management System. 
It is designed to manage students, courses, instructors, departments, and enrollments.

The system demonstrates various SQL concepts such as:
- Database creation
- Table relationships
- Primary and foreign keys
- CRUD operations
- Joins
- Subqueries
- Aggregation functions
- Window functions
- CASE statements
- Date functions

# Database Structure

The project contains the following tables:

1. Students
   Stores student personal and enrollment information.

2. Courses
   Stores course details and credit information.

3. Instructors
   Stores instructor details and salary information.

4. Departments
   Stores department names.

5. Enrollments
   Manages student-course relationships.

 # Features Implemented

- Insert, Update, Delete and Retrieve data
- Filtering students based on enrollment year
- Department-wise course filtering
- Counting students per course and department
- Inner Join and Left Join operations
- Subquery implementation
- Running total calculation using window function
- Student level classification using CASE expression
